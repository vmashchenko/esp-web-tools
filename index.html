Step-by-Step Instructions:

  1. Connect your LED device to the computer via USB cable (use a data cable, not just a charging cable)
  2. Select the firmware version from the list above (16-Band or 21-Band)
  3. Click the "CONNECT" button that appears below
  4. In the popup window, select your device port (it will look like cu.whcusbserial1410 or similar)
  5. Click "Install Spectrum Analyzer XX-Band" (where XX is the number of bands you selected)
  6. When prompted to erase the device, skip it (we don't need to erase the device)
  7. Click "Next" to start the installation
  8. Wait for the process to complete (up to 2 minutes) - you'll see a success message when done
Troubleshooting (click to expand)

If the update doesn't complete or gets stuck:

  • Try a different USB cable - make sure it's a data cable, not just a charging cable
  • Avoid using USB-C to USB-C cables - they often don't work well for flashing
  • Use a standard USB-A (wide) to USB-C cable, like the ones that come with power banks
  • Try a different USB port on your computer
  • Make sure you're using Chrome/Edge 89+ or Opera 76+ browser
  • If you get a permission error, make sure to allow serial port access when prompted

Note: The flashing process typically takes 1-2 minutes. If it takes significantly longer, there's likely an issue with the connection.
